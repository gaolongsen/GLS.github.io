###### 在此之前已在 GitHub Pages 搭建好了博客，且能够通过 github.io 进行访问。

* * *

#### 下面介绍一下如何将 github.io 与域名绑定：

##### 1\. 购买域名

可以在阿里云等购买。

##### 2\. 在 GitHub Pages 仓库添加 CNAME 文件并填入将绑定的域名

![在这里插入图片描述](https://img-blog.csdnimg.cn/2020090413304028.png)  
![在这里插入图片描述](https://img-blog.csdnimg.cn/20200904133102274.png)

![在这里插入图片描述](https://img-blog.csdnimg.cn/20200904133147320.png?1x-oss-process=image/watermark,type_ZmFuZ3poZW5naGVpdGk,shadow_10,text_aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L3dlaXhpbl80NTk2MTc3NA==,size_16,color_FFFFFF,t_70#pic_center)

> 填入的域名不要包含 `https://` 和 `www`。

##### 3\. 进入 GitHub Pages 仓库的 Settings 界面

![在这里插入图片描述](https://img-blog.csdnimg.cn/20200904133402828.png?1x-oss-process=image/watermark,type_ZmFuZ3poZW5naGVpdGk,shadow_10,text_aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L3dlaXhpbl80NTk2MTc3NA==,size_16,color_FFFFFF,t_70#pic_center)

##### 4\. 下拉至 GitHub Pages 模块并在 Custom domain 部分填入域名地址并保存

![在这里插入图片描述](https://img-blog.csdnimg.cn/20200904133445487.png?1x-oss-process=image/watermark,type_ZmFuZ3poZW5naGVpdGk,shadow_10,text_aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L3dlaXhpbl80NTk2MTc3NA==,size_16,color_FFFFFF,t_70#pic_center)

##### 5\. 添加域名解析

在命令行窗口 `ping` 你的 `user.github.io` 域名，得到一个IP地址，如图：

![在这里插入图片描述](https://github.com/JackTony123/picx-images-hosting/raw/master/Screenshot-from-2024-05-27-23-34-31.9dcsp2gwmx.webp)  
添加域名解析记录，此处以阿里云为例，添加两个 A 记录，用 ping 所得 IP，一个主机记录为：“www”，另一个为“@”。  
![在这里插入图片描述](https://img-blog.csdnimg.cn/20200904134110160.png#pic_center)  
稍等几分钟，此后通过 xxx.com 和 www.xxx.com 都能进行访问了。
